# Food Delivery Data Analysis

## Overview
This project merges food delivery data from multiple sources (CSV, JSON, and SQL) using Python and Pandas. The final dataset is created using LEFT JOIN operations and used to answer analytical questions.

## Datasets Used
- orders.csv (order details)
- users.json (user information and membership)
- restaurants.sql (restaurant details including cuisine and rating)

## Tasks Performed
- Loaded data from CSV, JSON, and SQL formats
- Merged datasets using LEFT JOINs
- Created a final merged dataset
- Answered analytical and conceptual questions

## Output Files
- final_food_delivery_dataset.csv
- food_delivery_data_analysis.ipynb
